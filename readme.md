# Claude Design Skill

> 7 design systems merged into one compressed SKILL.md.
> Load once. Ship zero slop. Save ~98% on design prompt tokens.

---

## What this is

A single `SKILL.md` file that merges:
- Emil Kowalski animation physics
- Impeccable design vocabulary + 17 slash commands
- /polish pre-ship checklist
- Taste skill anti-slop rules + Bento 2.0
- Self-Made Designer principles
- Framer Motion (motion/react) best practices
- UI/UX Pro Max token architecture

Compressed using Caveman Compression — strip grammar,
keep facts, save tokens. Every rule is actionable code
or a hard constraint.

---

## Quick start

### Option 1 — Claude Project (recommended)
1. Go to Claude.ai → Projects → New Project
2. Upload `skills/claude-design/SKILL.md` to Knowledge
3. Use this prompt in every chat:

```
Apply our design skill and create [X]
```

### Option 2 — Upload per session
Attach `SKILL.md` to the first message in any Claude chat.

### Option 3 — Slash commands (once skill is loaded)
```
/polish    /animate    /audit    /bolder
/quieter   /distill    /harden   /overdrive
```
